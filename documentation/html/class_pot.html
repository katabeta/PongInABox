double Pot::getNormalizedRead ( void  )
private

normalizes the raw reading from the potentiometer.

It ranges the output from 0 to 1, effectively giving the percentage. See: http://stats.stackexchange.com/questions/70801/how-to-normalize-data-to-0-1-range for more details. Feeds the getLEDNumRead() method.
